## 가변배열

-  행마다 열의 갯수를 다르게 잡을 수 있다.

  new 타입 [행의크기] [ ]

ex) int[ ] [ ] emp = new int [5] [ ];

​	emp[0] = new int [10];

​	emp[1] = new int [20];

​	emp[2] = new int [50];

이러한 방식으로 행마다 필요한 열의 갯수를 잡을 수 있다.



Java 5, JDK 1.5 이후부터 나온 for문



## for each 문

- 배열 또는 컬렉션 객체의 데이터들을 꺼내서 반복처리하려는 경우 사용하는 반복문.

- 앞에서부터 차례대로 하나하나 꺼내서 처리하려는 경우(역순 안됨)

  for(변수선언 : 배열 또는 컬렉션 객체)

  ​		반복문장;

  ex) for(int data : score)

  ​			sum += data;

  score 배열이 갖고있는 데이터를 반복

  score 배열이 갖고있는 데이터를 모두 더하라

  일반적인 for문보다 속도가 빠르다.
